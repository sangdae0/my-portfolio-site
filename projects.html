  2. 뉴욕 택시 탑승 데이터 분석 프로젝트
    • 뉴욕 택시 탑승 데이터를 활용한 학교 Data Analysis 기말 자유 형식 과제 입니다.
    • 뉴욕 택시 안에서의 광고가 과연 관광객에게 더 효과적인지, 현지인에게 더 효과적인지가 주제입니다.
      • 트립 어드바이저와 구글 맵을 가지고 유명한 레스토랑(관광객 보다 현지인이 더 찾을 거라고 가정)과 관광지(현지인 보다 관광객이 더 찾을 거라고 가정) 목록을 만든 후, 격자로 나눈 택시 하차 히트맵을 레스토랑 히트맵과 관광지 히트맵에 대조시켜 어떤 게 더 인기가 많은 지 체크했습니다.
      • 결론은 관광객쪽이 더 많이 사용하는 것 같으니 관광객을 대상으로 한 광고가 더 효과가 많을 것 같다 였지만 그렇게 극명한 차이가 나지 않아 다른 분석 방법을 시도하는 게 좋았을 것 같습니다.
        • 오히려 택시 하차 데이터를 가지고 히트맵을 생성하는 과정에서 히트맵이 탑승객들이 주로 맨하탄으로 향하는지, 그리고 맨하탄 중심부에서 내리는지를 파악할 수 있어서 재밌었던 기억이 있습니다.
    • Python, Pandas, NumPy, Google Map, Jupyter Notebook을 활용했습니다.
    • 코드 샘플 - 주피터 노트북 파일
  3. L-System을 이용한 트리 모델 생성기와 이를 활용한 게임(제목: Bullet Screw Plus Tree) 제작 프로젝트

      • 상수와 파라미터로 이루어진 표현식을 처리해 다양한 트리 모델 생성이 가능합니다.

  4. OpenGL 프로젝트
